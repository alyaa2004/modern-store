 🛒 Modern Store

A modern full-stack e-commerce web application built using the MERN Stack.

🚀 Live Demo

https://modern-store-gamma.vercel.app/

 📂 GitHub Repository

https://github.com/alyaa2004/modern-store

---

 ✨ Features

- User Registration & Login
- JWT Authentication
- Password Encryption (bcrypt)
- Product Search
- Product Details
- Shopping Cart
- Responsive Design
- REST API
- MongoDB Atlas

---

🛠 Tech Stack

- React.js
- Material UI
- Redux Toolkit
- React Router
- Express.js
- MongoDB Atlas
- JWT
- bcrypt
- Axios
